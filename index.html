SOFTWARE & WEB BASED PLATFORM

Adobe Photoshop
Saya membuat design di software bernama adobe photoshop.
Di software ini kita bisa mengedit gambar maupun mendesign poster, karna tools nya lengkap
dan saya biasanya membuat design atau mengedit gambar jika mood senang saya sedang naik,
kadang mood saya pernah turun dan itu mengakibatkan Art block jadi tidak bisa mendesign.
Di photoshop ini kita dapa menyalurkan kreatifitas kita menjadi sesuatu yang nyata

Adobe Illustrator
Software berbasis vektor yang memungkinkan desain tetap tajam meski diperbesar
berkali-kali. Pertama kali dirilis pada tahun 1987, Illustrator terus berkembang dan menjadi alat
utama para desainer Banyak logo terkenal dunia, seperti Apple dan Coca-Cola,
dibuat menggunakan Illustrator karena kemampuannya dalam menghasilkan grafis berkualitas tinggi.

Visual Studio Code
Visual Studio Code adalah editor kode sumber ringan yang dirilis oleh
Microsoft pada tahun 2015 dan sangat populer di kalangan pengembang.
Dikenal dengan antarmuka yang ramah pengguna dan
kustomisasi yang luas, VS Code mendukung berbagai bahasa pemrograman seperti HTML, CSS, dan JAVASCRIPT.

Github
GitHub adalah platform berbasis web yang dirancang untuk pengembangan dan kolaborasi kode,
dirilis pada tahun 2008 dan sekarang dimiliki oleh Microsoft.
Menggunakan Git sebagai sistem version control, GitHub memudahkan pengembang untuk menyimpan,
melacak, dan berkolaborasi dalam proyek pemrograman secara efisien.
